# Color Harmony Net

一个基于深度学习的图像颜色和谐度评估模型。

## 功能特点

- 使用 LDA 进行颜色主题建模
- 支持 RGB 和红外图像的特征提取
- 实时可视化训练过程
- 提供预训练模型

## 安装

1. 克隆仓库:
```bash
git clone https://github.com/your-username/color-harmony-net.git
cd color-harmony-net
```

2. 安装依赖:
```bash
pip install -r requirements.txt
```
